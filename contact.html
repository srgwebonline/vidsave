Frequently Asked Questions

How long does it take to get a response?

We aim to respond to all inquiries within 24-48 hours during business days. Technical support queries are prioritized and typically receive responses within 24 hours. Weekend inquiries may take until the next business day.

Do you offer technical support for tool issues?

Yes, we provide free technical support for all our tools. Please include specific details about the issue, including:

  • Browser and version
  • Steps to reproduce the issue
  • Screenshots or error messages
  • URL where the issue occurred
Can I request a new feature for the toolkit?

Absolutely! We welcome all feature suggestions. Please send detailed proposals to features@shortscreator.tools. Include information about:

  • The problem you're trying to solve
  • How you envision the feature working
  • Any similar features on other platforms
  • Why this would benefit content creators
Are there any costs for using your tools?

No, all our tools are completely free to use. We're supported by advertisements and optional donations. We will never charge for basic tool functionality. Some advanced features may be added in the future with premium options, but core tools will remain free.

Do you offer API access for developers?

Currently, we don't offer public API access. However, we're working on developer-friendly solutions. If you're interested in API access for integration purposes, please contact partnerships@shortscreator.tools with your use case and requirements.

How can I report a bug or technical issue?

Please report bugs to bugs@shortscreator.tools with the following information:

  • Detailed description of the issue
  • Steps to reproduce the problem
  • Browser, version, and operating system
  • Screenshots or screen recordings
  • Console error messages (if any)

We prioritize bug fixes and aim to resolve critical issues within 48 hours.
